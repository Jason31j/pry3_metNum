Dependencias:
•	librería Numpy, esta es fundamental para poder ejecutar el proyecto, ya que fue necesaria para la resolución de los sistemas de ecuaciones y manejo de matrices; no viene de serie, debe ser instalada.
•	Librería Math, esta ejecuta su función en el cálculo de logaritmos y raíces cuadradas, es una función preinstalada, viene con el paquete de instalación de Python.
El archivo clases.py, contiene las clases y los métodos necesarios para la resolución de la regresión en los 3 tipos propuestos para este proyecto, regresión lineal, regresión polinomial de 2do grado y regresión exponencial.
En el archivo main.py solo esta las instrucciones para leer los datos de las variables dependiente e independiente, luego de eso instanciar los objetos con las clases definidas en el archivo clases.py y luego llamar a los métodos para resolver el problema.
